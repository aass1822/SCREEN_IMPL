            position:relative;
        }

        }

# 하루 정리
> 선택자  
*: 전체 선택자  
div,span : 요소선택자  
.class1 : 클래스 선택자  
#id1 : id선택자  
,: 그룹선택자  

> > : 자식 선택자  
: 자손 선택자  

> ~ : 동위 선택자(아래 n)  
+ : 동위 선택자(아래 1) -> 체크박스   

> :hover : 의사선택자(마우스오버)    

> ::before : 가상선택자  
::after : 가상선택자 -메뉴 구분선만들기(다음시간)  

> nth-child : n번째 자식요소
