# 자바스크립트
자바스크립트(JavaScript)는 웹 페이지에 동적인 기능을 추가할 수 있도록 설계된 인터프리터 방식의 프로그래밍 언어  
> 데이터 기반의 자료를 저장 처리  
서버 - 제공 클라이언트 -요청 (웹 서비스):프로그램  
다양한 이벤트를 처리해주는 브라우저에 있는 대부분의 기능을 쓸 수 있게함  

> 인터프리터 언어 : 실행전에 컴파일이 필요없는 언어
동적 타이핑 : 변수의 타입이 실행중에 결정됨
객체 기반 : 거의 모든 것이 객체로 구성됨
이벤트 기반 : 사용자 상호작용에 따라 코드실행
브라우저 내장 : 별도 설치 없이 웹 브라우저에서 바로 실행가능  
크로스 플랫폼 : 운영체제에 상관없이 대부분의 환경에서 실행 가능
