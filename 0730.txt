# 네트워크
> 기본개념 -> 1.6 -> 3.3 (만 공부한다고 함)
1. 네트워크(NETWORK) : 데이터 교환을 위해 통신장치를 연결한 통신망
---
네트워크 토폴로지
토폴로지란 네트워크의 물리적 또는 논리적 배치 구조를 의미  
각 토폴로지는 고유한 장점과 단점을 가지고 있다.
1. 버스 토폴로지 : 모든 장치가 하나의 중앙 케이블에 연결
2. 스타 토폴로지 : 모든 장치가 중앙 허브나 스위치에 연결
3. 링 토폴로지 : 각 장치가 양옆의 두 장치와 연결되어 원형을 이룸
4. 메시 토폴로지 : 모든 장치가 서로 직접 연결되는 완전 메시와 일부만 연결되는 부분 메시가 있다.
LAN (Local Area Network)
1. 좁은지역 (건물,캠퍼스) 내 네트워크
2. 높은 데이터 전송속도 (1-10 Gbps)
3. 단일조직이 소유 및 관리
4. 컴퓨터실,사무실 네트워크
WAN (Wide Atra Network) / 연결해주는 장치 라우터
1. 넓은 지역 (도시,국가,대륙 간) 연결
2. 상대적으로 낮은 데이터 전송속도
3. 통신사업자의 서비스 이용 (임대)
4. 예 : 인터넷,기업의 지사 간 네트워크
프로토콜 : 네트워크에서 데이터를 교환하기 위한 규칙과 약속의 집합  
사람의 언어와 같은 역할
구조 : 구문 - 의미 - 타이밍
예시 : 구문) 오늘 | 수업 | 마치고 | 밥 | 먹으러 갈까? 문장 완성을 위한 문법체계  
        의미) '오늘','내일','모래','주말','밥','커피'.. 단어
      타이밍) 내용을 전달하는 속도나 순서 밥( 밥 오늘 수업 갈까? - x 대답을 하지 않는다..시간초과
네트워크 모델 : 컴퓨터나 장치들이 서로 데이터를 주고받을 수 있도록 구성한  
통신 구조 또는 계층 체계 (OSI)
OSI 계층구조 : 1계층:물리계층 2계층:데이터 링크 계층 3계층:네트워크 계층 4계층: 전송 계층(중요)
5계층:세션 계층 6계층:표현 계층 7계층:응용 계층

> 문제 정리
1. Network 는 무엇이며 무엇을 목적으로 하는가?
데이터 교환을 위한 통신장치를 연결한 통신망
2. Network 의 종류는 크게 ooo/ooo 로 나눌 수 있다
WAN LAN
3. LAN 의 특징에 대해서 서술하세요
좁은지역에 많이 쓰고 데이터 전송 속도가 빠르다
4. WAN의 특징에 대해서 서술하세요
넓은지역 (국가,지역) 상대적 느린 데이터 전송 속도
5. InterNet 이란
전 세계의 공용 네트워크. 누구나 접근 가능 WAN + LAN을 묶은 거대 통신망
6. 프로토콜은 무엇인가?
네트워크 환경에서 통신을 위해 사용하는 규칙의 집합
데이터 포맷, 전송 순서, 에러 제어 방식 등 포함  
Syntext + meaning + timing  
7. 네트워킹 모델의 목적은 무엇인가?  
효율적인 네트워크 설계, 문제 발생 시 원인 파악 용이  
OSI&TCP/IP : 이기종간의 통신을 위해서 제작한 표준 지침서  
8. TCP/IP 모델과 OSI 7 모델의 계층을 적고 각 계층별 역할을 적으세요
OSI 모델
1계층: 물리계층 = 실제 장치들을 연결하기 위한 전기적, 물리적 세부 사항 정의 (핀 배열, 전압, 전선 명세 등)  
2계층: 데이터 링크 계층 = 포인트 투 포인트(Point to Point) 간의 데이터 전달 방식 결정    
3계층: 네트워크 계층 = 경로 탐색 (라우팅)  
4계층: 전송 계층(중요) = 종단 간 데이터 신뢰성 보장 / 송수신지간(End to End) 데이터의 운송방식 결정  
5계층: 세션 계층 = 통신 세션 설정 및 유지  지속적인 데이터 교환을 위해 대화형 채널 생성  
6계층: 표현 계층 = 데이터 형식 변환, 암호화, 압축  
7계층: 응용 계층 = 사용자 인터페이스 제공 / 통신을 수반하는 프로그램(사용자/장치에 의해 작업물 생성)  
TCP/IP 모델
1계층: Network Interface 계층 = 물리적인 네트워크 매체를 통해 데이터를 전송하고, 데이터 링크를 설정하고 유지하며, 장치 간의 데이터 링크를 설정합니다.   
2계층: 인터넷 계층 = 데이터를 패킷 단위로 분할하고, IP 주소를 사용하여 최적의 경로를 찾아 데이터를 전송합니다.   
3계층: 전송 계층 = 데이터의 신뢰성 있는 전송을 담당  
4계층: 애플리케이션 계층 = 사용자에게 직접 인터페이스를 제공하며, 다양한 응용 프로그램들이 이 계층에서 동작
