# Routing Protocol
> Route : 경로, 길
Router : 경로안내 장치
Routing : 경로안내중 / 경로안내작업
Protocol : 미리 설치된 통신 프로그램
RP : 경로 안내 작업 통신 프로그램
└ 정적 : 목적지에 대한 방향정보를 관리자가 직접 넣어주는 방식 (불편함, 라우터 성능 낮아도)
└ 동적 : 목적지와 방향정보를 프로그램이 계산하여 넣어주는 방식 ( 편함, 라우터 성능 높아야)
	└전체경로학습 (선행학습 필요)
	└최적경로계산 (계산 알고리즘으로 계산)
└정적 - static 단일 목적지에 대한 방향정보를 관리자가 직접 넣어줌
default 그외 나머지 모든 방향정보를 관리자가 직접 넣어줌
